# secureweb
Cyber Crime Awareness Portal
A simple, responsive, and modern college project built with Python (Flask), SQLite (Database), and HTML/CSS/JavaScript (Bootstrap 5).

This portal aims to educate users on common cyber crimes (Phishing, UPI Fraud, QR Code Scams, Social Media Scams, and Online Shopping Fraud), provide safety guidelines, offer an interactive awareness quiz, and include an Incident Report form with a basic Admin Panel to manage submissions.

Features
Home Page: Modern tech-themed hero section, brief overview, and cyber security statistics.
Threat Intelligence: Interactive tab-based layout exploring different cyber crimes, how they work, and prevention checklists.
Safety Checklist: Clean infocard layout with the 6 rules of cyber hygiene (OTP, Passwords, 2FA, etc.).
Interactive Quiz: JavaScript-driven 10-question multiple-choice assessment with a progress indicator, instant explanation feedback, and dynamic scoring advice.
Fraud Reporting: A validating web form to register scam details, directly writing to the database.
Admin Control Panel: View, filter, and delete reports, secured with standard session-based user authentication.
